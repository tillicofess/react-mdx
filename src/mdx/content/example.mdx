# 欢迎使用MDX运行时编译

这是一个展示如何在React应用中**运行时编译MDX内容**的示例。

## 主要特性

- ✅ 运行时MDX编译
- ✅ 支持Frontmatter解析
- ✅ 自定义组件样式
- ✅ 代码高亮显示
- ✅ 响应式设计

## 代码示例

下面是一个简单的React组件示例：

```tsx
function Welcome({ name }) {
  return (
    <div className="welcome">
      <h1>Hello, {name}!</h1>
      <p>欢迎使用MDX!</p>
    </div>
  );
}
```

## 技术栈

本示例使用了以下技术：

1. **@mdx-js/mdx** - MDX编译器
2. **@mdx-js/react** - React MDX提供者
3. **React 19** - 用户界面库
4. **Vite** - 构建工具
5. **TypeScript** - 类型安全
